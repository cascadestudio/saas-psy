# SaaS Psy - Web App

Next.js frontend application for psychologists and patients.

## Development

```bash
# From monorepo root
npm run dev

# Or from this directory
npm run dev
```

## Environment Variables

Copy `.env.example` to `.env.local` and configure the values.

## Features

- Psychologist dashboard
- Questionnaire management
- Patient interface for completing questionnaires
- Real-time scoring and interpretation
- Email notifications
